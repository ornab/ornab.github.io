AN AWESOME DEVELOPER ON PLANET

Hi, I am Hasan Zamil Ahmed. I have completed my gradutation from a reputed public University on Computer Science. But my introduction to programming was a bit later when I was admitted to CSE (I had a dream to become a Mechanical Engineer or an Astronaut). But after getting admission in CSE soon I had realize that this is the right place for me.

My first introduction to programming languages was with C, the mother of all programming languages. I have learned C for about 1 year & attended some programming contests arranged by my depertment & also in online, by this time I have learned basic Data Structure & Algorithm. Then I swicthed to C++ & it was my first introduction to Object Oriented Programming. After some time, I met Java, my first actual love. I was first entered to the world of Web Development by Java web technologies, Jsp & Servlet. I have done some projects with them.

If you are still reading, you are awesome. By the end of my 4th Semester, I first heared about PHP. I leaned the basic & Object Oriented PHP for about 6 months & also enhanced my frontend skills by doing PSD to HTML & playing with Javascript staff. I have learned Wordpress & have done a course on it under Rasel Ahmed in RR Foundation. I have done many projects on Wordpress.

In my 6th semester, I came to know about a magic, called Laravel. This is the only reason for which I'm still in the PHP world. I have a premium Subscription in Laracasts & learned a lot from my Idol Jeffrey Way. I have done a course on Laravel under Sumon Molla Selim vai. Also done course on Become a Master in Laravel in Udemy. I have done many projects on laravel which are showcased in the Portfolio section.
